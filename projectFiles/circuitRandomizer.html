Circuit Randomizer

February 2024 - Current

MercerX Lab Research Project

HTML, CSS, JavaScript



The goal of this project is to create a circuit randomizer that can continuously provide students with random circuits in order to test their ability to do circuit analysis.

My role in this project is to help create the interface and get various functionalities implemented (such as but not limited to: the step-by-step guide for how to solve the given circuit, the matrix interface for calculations, and fixing bugs/issues).

This project is intended for classroom and laboratory use.



This project is supervised by Professor Shayla Sawyer, and the MercerX Lab at Rensselaer Polytechnic Institute.
